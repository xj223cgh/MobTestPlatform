# Preferences

> Note: The preferences module has been removed. All configuration items are now hardcoded in the code.

## Removed Features

The following features have been removed from the application:

- Audio Control
- Audio/Video Recording
- Window Control
- Video Control
- Device Control
- Scrcpy Parameter Configuration
- Gnirehtet Parameter Configuration
- Gnirehtet Path Configuration
- Gnirehtet Fix Function
- Window Arrangement Function

## Remaining Features

The application now retains only core functionality, with configuration items hardcoded. No additional configuration is required from users.

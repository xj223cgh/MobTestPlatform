# 窗口编排功能

> 注意：窗口编排功能已被移除。

## 功能状态

窗口编排功能已从 Escrcpy 应用中移除。该功能原用于通过可视化拖拽界面管理多个设备窗口的布局，包括：

- 同时管理多个设备窗口的布局
- 通过拖拽方式调整窗口位置和尺寸
- 设置全局默认配置和设备特定配置
- 实时预览窗口布局效果
- 保存和加载自定义布局方案

## 移除原因

为了简化应用结构和提高性能，项目决定移除窗口编排功能，专注于核心的设备镜像和控制功能。

## 替代方案

设备窗口的位置和大小现在由系统自动管理，或通过 scrcpy 命令行参数控制。如需自定义窗口行为，建议直接使用 scrcpy 的原生命令行参数。
